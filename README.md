# 🚀 Portfolio - Pauline Ogriné

> **Un portfolio moderne présentant mon parcours d'ingénieure optique en transition vers le machine learning et le développement full stack.**

Bienvenue sur mon portfolio personnel ! Ce site présente mon parcours en tant qu'ingénieure optique, mes projets et mes compétences en développement full stack et machine learning.

## 📋 À propos

Je suis une ingénieure diplômée en optique qui s'oriente vers le **machine learning** et le **deep learning** appliqués au traitement d'images. Je développe des pipelines de traitement et d'analyse de données.

## 🛠️ Technologies

- **Frontend** : Vue.js 3, TypeScript, Vite
- **Styling** : CSS avec designs modernes et responsifs
- **Outils** : Git, Docker, Linux

## 📦 Installation

```sh
npm install
```

## 🚀 Développement

```sh
npm run dev
```

Lance le serveur de développement sur `http://localhost:5173`

## 🏗️ Build

```sh
npm run build
```

Compile et minifie le projet pour la production.
